# Contributing to 60-Day English Curriculum

Thank you for wanting to improve this resource for Hindi-speaking English learners! 🙏

## Ways to contribute

### 1. Report errors
If you find a grammar mistake, wrong Hindi translation, or unclear explanation — please open an issue describing:
- Which file and section
- What's wrong
- Suggested fix (if you have one)

### 2. Add a new grammar chapter
We'd love chapters on:
- Articles (a, an, the)
- Prepositions (in, on, at, for, with)
- Conjunctions and connectors
- Tenses (present perfect, past continuous, etc.)
- Question formation
- Phrasal verbs
- Idioms for office use

**Format to follow** (see `grammar/modal-verbs.md` or `grammar/v2-verb-forms.md` as templates):
- Start with a clear explanation + Hindi comparison
- Include a reference table
- Show common mistakes (❌ Wrong / ✅ Right format)
- Add office-specific example sentences
- End with a short practice section or study plan

### 3. Translate for other languages
If you speak Tamil, Bengali, Telugu, Marathi, Punjabi, Gujarati, or any other Indian language and want to adapt this curriculum with comparisons in your language — please open a discussion first so we can coordinate folder structure (e.g., `tamil/`, `bengali/`).

### 4. Add practice material
- More office email templates
- More speaking practice scripts
- More vocabulary lists with Hindi meanings
- More quiz questions

### 5. Improve existing content
- Better examples
- More natural office phrases
- Updated free resource links (YouTube channels, apps, websites change over time)

## Style guidelines

- Keep explanations simple — avoid heavy grammar jargon
- Always include Hindi comparison where it helps (use Devanagari script, not transliteration, where possible)
- Use real office/daily-life examples, not textbook-style abstract sentences
- Mark mistakes clearly with ❌ and corrections with ✅
- Keep tone encouraging and practical — this is for confidence-building, not academic perfection

## How to submit

1. Fork the repository
2. Create a new branch (`git checkout -b add-prepositions-chapter`)
3. Make your changes
4. Submit a pull request with a clear description of what you added/fixed

## Questions?

Open an issue or start a discussion — happy to help coordinate larger contributions like translations or new sections.
