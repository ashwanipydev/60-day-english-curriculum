## Today's Speaking Challenge

> Speak about a problem you solved recently

---

## Problem-Solution Language is Essential

At work, you constantly need to describe: what went wrong, what you did about it, and what the result was.

---

## Problem Description Language

**Identifying the problem:**
- "We ran into an issue with..."
- "There was a complication in..."
- "We noticed that [X] wasn't working as expected."
- "The challenge we faced was..."

**Explaining the impact:**
- "This caused a delay in..."
- "As a result, we couldn't..."
- "This affected [who/what]."

---

## Solution Language

**What you did:**
- "We decided to..."
- "I took the initiative to..."
- "We approached it by..."
- "The solution we came up with was..."

**The result:**
- "As a result, we were able to..."
- "This helped us..."
- "The outcome was positive — we managed to..."
- "In the end, [the issue was resolved / we met the deadline / the client was satisfied]."

---

## The Problem-Solution Framework

Use this structure:

1. **Background:** "We were working on [X] when..."
2. **Problem:** "We encountered [issue] which caused [impact]."
3. **Action:** "I/We addressed this by [steps taken]."
4. **Result:** "The outcome was [result]."
5. **Learning:** "What I learned from this is [lesson]."

---

## Speaking Task

Think of a real problem you solved recently at work (or anywhere). Tell the story using the framework above. Speak for **2 minutes**. Record yourself.
