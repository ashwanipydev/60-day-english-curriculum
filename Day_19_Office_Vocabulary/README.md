## 20 Essential Office Terms

Master these 20 words — they appear in almost every workplace conversation.

| Term | Hindi | Example |
|------|-------|---------|
| **deadline** | अंतिम तिथि | "The deadline is Friday." |
| **deliverable** | कार्य परिणाम | "The deliverable is a 10-page report." |
| **stakeholder** | हितधारक | "We need to update the stakeholders." |
| **bandwidth** | समय/क्षमता | "I don't have the bandwidth right now." |
| **bottleneck** | रुकावट | "The approval process is a bottleneck." |
| **align** | सहमत होना | "Let's align on the plan." |
| **escalate** | आगे बढ़ाना | "I'll escalate this to the manager." |
| **delegate** | सौंपना | "I'll delegate this task to Ankit." |
| **sign off** | मंजूरी देना | "The manager needs to sign off." |
| **loop in** | शामिल करना | "Can you loop in Priya?" |
| **take offline** | अलग बात करना | "Let's take this offline." |
| **action item** | करने का काम | "My action item is to send the report." |
| **sync** | मिलना | "Let's sync tomorrow at 10." |
| **blocker** | रुकावट | "There's a blocker I need help with." |
| **scope creep** | काम का बढ़ना | "We need to avoid scope creep." |
| **EOD** | दिन का अंत | "I'll send it by EOD." (End of Day) |
| **EOW** | सप्ताह का अंत | "Report by EOW please." (End of Week) |
| **KPI** | मुख्य संकेतक | "Our KPIs are on track." |
| **follow up** | आगे चेक करना | "I'll follow up on this." |
| **touch base** | संपर्क करना | "Let's touch base this afternoon." |

---

## Formal vs Informal Office Language

| Informal | Formal |
|----------|--------|
| "Can you do this?" | "Could you please handle this?" |
| "When's the meeting?" | "When is the meeting scheduled?" |
| "I need this ASAP." | "This is urgent — could you prioritize it?" |
| "Talk later." | "I'll follow up with you." |
| "My bad." | "I apologize for the oversight." |

## Speaking Task

Use **5 of today's 20 terms** in sentences about your current work. Speak them out loud, using them naturally in context.
